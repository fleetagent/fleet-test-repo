# fleet-test-repo

Dummy target repo for Fleet Phase 1B walking-skeleton tests. Not a real application.

## Scripts

- `npm test` — runs `node --test` against `test/**/*.test.js`
- `npm run lint` — trivial no-op for now (`node -e "console.log('lint ok')"`)

## Why this repo exists

Agents being developed in the Fleet codebase need a harmless target to push test PRs to. This is it. Delete arbitrary branches without guilt.
